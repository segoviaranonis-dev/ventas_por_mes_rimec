# CÓMO EJECUTAR EL SISTEMA

## Sistema Principal (Importadora - Streamlit)
1. Abrir terminal en: C:\Users\hecto\Documents\Prg_locales\ventas_por_mes_rimec-main
2. Ejecutar: streamlit run main.py
3. Abrir navegador en: http://localhost:8501

## Bazar Web (clientes finales - Next.js)
1. Abrir terminal en: C:\Users\hecto\Documents\Prg_locales\bazzar-web
2. Ejecutar: npm run dev
3. Abrir navegador en: http://localhost:3000

## Web RIMEC (vendedores mayoristas - Next.js)
1. Abrir terminal en: C:\Users\hecto\Documents\Prg_locales\rimec-web
2. Ejecutar: npm run dev
3. Abrir navegador en: http://localhost:3001

## RESUMEN DE PUERTOS
- localhost:8501 → Streamlit (administración RIMEC)
- localhost:3000 → Bazar Web (clientes finales)
- localhost:3001 → RIMEC Web (vendedores mayoristas)

## Para ejecutar los 3 al mismo tiempo
Abrir TRES terminales separadas y ejecutar uno en cada una.
